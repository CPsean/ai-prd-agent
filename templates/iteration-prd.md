# 迭代 PRD 模板（Iteration PRD）

> **使用场景**：对已有功能的局部优化、扩展或修复，重点描述"变了什么"，不重写整个功能。
> **复制方法**：运行 `/new-prd feature [标题]` 并在询问时选择"迭代类需求"，AI 自动从此模板创建。

---

```yaml
---
type: iteration-prd
id: F-XXX
title: [标题]
status: draft
version: "1.0"
created: YYYY-MM-DD
updated: YYYY-MM-DD
author:
feature-area:
modifies: F-XXX           # 被本次迭代修改的原始 PRD ID
modifies-title: [原始功能名称]  # 原始 PRD 标题，便于人工识别
has-prototype: false
prototype-path:
has-page-spec: false
page-spec-path:
---
```
