# 功能 PRD 模板（Feature PRD）

> **使用场景**：一个完整功能模块，面向设计、研发、测试多个受众。
> **复制方法**：运行 `/new-prd feature [标题]`，AI 自动从此模板创建。

---

```yaml
---
type: feature-prd
id: F-XXX
title: [标题]
status: draft
version: "1.0"
created: YYYY-MM-DD
updated: YYYY-MM-DD
author:
feature-area:
epic:             # 所属史诗 PRD 的 ID，如 E-001
related-stories:  # 关联的 Story Card，如 [SC-001, SC-002]
has-prototype: false
prototype-path:   # 如有原型：prototype/index.html
---
```
