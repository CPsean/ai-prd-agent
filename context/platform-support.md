## 初始化说明

请在首次使用前填写支持端清单。完成后删除本段说明。

**状态说明**：
- ✅ 支持：功能完整可用
- ⚠️ 部分支持：有功能限制，需在 PRD 中说明差异
- ❌ 不支持：当前版本不覆盖
- 🔄 规划中：未来版本计划支持

## PRD 中引用方式

在 PRD 的"端差异说明"章节，只列出本功能实际存在差异的端：

```markdown
## 端差异说明

> 支持端清单参考：context/platform-support.md
> 以下仅列出存在差异的端，未列出的端与 Web 端行为一致。

| 端 | 差异说明 |
|----|---------|
| Web 端 | 基准实现，支持完整功能 |
| 微信小程序 | 不支持文件直传，需通过中转接口上传 |
| 钉钉端 | 需使用钉钉 JSAPI 调起签名组件，交互路径不同 |
```

若本功能全端行为一致，填写：

```markdown
## 端差异说明

本功能全端行为一致，无差异。
```
