# 命令：放弃草稿

主动放弃指定草稿，删除草稿目录并释放预注册 ID：

$ARGUMENTS

## 解析参数

从 `$ARGUMENTS` 中识别：
- **标题**：草稿的中文标题（必须，无则询问用户）

## 执行步骤

**1. 定位草稿**

按以下方式查找目标草稿：

a. 读取 `drafts/_draft-registry.md`，按标题匹配条目，获取 ID
b. 若预注册表中未找到，在 `drafts/` 目录下搜索包含标题的目录（兼容 `drafts/[ID]-[标题]/` 和 `drafts/[标题]/` 两种格式）
c. **均未找到**：提示"未找到草稿「[标题]」，请确认标题是否正确"，停止执行

**2. 二次确认（必选，不可跳过）**

输出确认提示：

> **确认要放弃草稿「[标题]」（ID：[ID]）吗？**
> 草稿目录 `drafts/[ID]-[标题]/` 将被删除，此操作不可撤销。
>
> 回复「确认」继续，或回复「取消」。

- 用户回复「取消」或其他非确认内容 → 输出"已取消，草稿保留"，停止执行
- 用户回复「确认」→ 继续执行

**3. 删除草稿目录**

删除 `drafts/[ID]-[标题]/` 整个目录（含所有子文件：prd.md、rdd.md、CHANGELOG.md、archive/ 等）。

**4. 清理预注册条目**

从 `drafts/_draft-registry.md` 删除对应 ID 的条目。

若预注册条目不存在（异常状态）：仍完成目录删除，输出提示"预注册条目不存在，目录已手动清理"。

**5. 输出确认**

> 草稿「[标题]」已放弃，目录和预注册 ID（[ID]）已清理。

## 注意

- 本命令仅操作 `drafts/` 中的内容，不影响 `prds/` 正式区
- 二次确认为必选步骤，不可跳过
- 被释放的 ID 可被后续新草稿重新使用
