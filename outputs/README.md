# outputs/ — 对外交付物

> 存放需要分享给外部（开发团队、业务方、客户）的最终文档。内容由用户主动整理进来。

---

## 适合放在这里的内容

- 评审演示材料、汇报文档
- 项目交接文档、操作手册
- 从 `drafts/` 整理完成的方案设计文档

> PRD 本身不流入此目录，`prds/` 是 PRD 的最终归宿。

---

## 流转来源

```
drafts/[方案文档]  →（用户确认整理）→  outputs/
```

**触发方式**：告诉 AI「把 [文件] 整理到 outputs」即可。

---

## 目录结构（自由组织）

```
outputs/
  2026-Q1-需求评审演示.md      ← 评审材料
  登录方案设计-交付版.md        ← 方案文档
  项目交接-用户模块.md          ← 交接文档
```

---

## 注意事项

- 此目录内容不被 AI 命令自动消费，不影响 PRD 问答和摘要的结果
- PRD 请在 `prds/` 中查阅，不要在此维护 PRD 副本
