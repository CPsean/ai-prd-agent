# 测试用例：技能管理（F-008）

> 关联 PRD：`prds/技能管理/prd.md`（F-008）
> 关联规范：`CLAUDE.md` 技能管理行为规范
> 触发方式：自然语言（无专用斜杠命令）
> 前置条件：`.claude/skills/` 目录存在

---

## TC-SM-01 自建技能创建 → skill-generate 引导完整流程

| 字段 | 内容 |
|------|------|
| **状态** | — |
| **功能编号** | SKL-INS-001 |
| **测试目标** | 验证 PM 发起创建技能时，AI 引导提供 name/description/内容，生成候选 SKILL.md 后自动进入质检 |
| **前置条件** | `.claude/skills/` 目录存在 |

**测试输入**
```
帮我创建一个竞品分析的技能
```

**预期行为**
1. AI 识别为技能创建意图
2. 引导 PM 提供：技能名称（如 `competitive-analysis`）、描述、触发条件、执行步骤、输出格式
3. 生成候选 SKILL.md（不直接写入 `.claude/skills/`）
4. 自动进入 SKL-QC-001 三级质检

**检查要点**
- [ ] AI 引导 PM 提供名称、描述和内容
- [ ] 候选文件未直接写入 `.claude/skills/`
- [ ] 生成后自动触发质检（无需 PM 额外操作）
- [ ] SKILL.md 包含 frontmatter（name + description）
