# 测试用例：/sync-docs 命令（F-006 文档同步）

> 关联 PRD：`prds/文档同步/prd.md`（F-006）
> 关联命令：`.claude/commands/sync-docs.md`
> 参数格式：`/sync-docs [标题或ID]`
> 前置条件：目标 PRD 已在 `prds/` 正式区或 `drafts/` 草稿区
> Codex 等效输入：「检查文档一致性：[标题]」或「帮我核查[标题]的关联文档」
